Über MeshNet Connect

MeshNet Connect ist ein dezentrales Bluetooth Mesh Netzwerk — ein lebendiges Experiment in Peer-to-Peer Kommunikation, das ohne Server auskommt.

Gedanke

Netzwerke sollten nicht von zentralen Punkten abhängen. MeshNet zeigt wie Menschen selbstorganisierende Netzwerke bauen können — direkt über Bluetooth, ohne Infrastruktur.

Eigenschaften

Dezentral

Kein Server, keine zentrale Infrastruktur. Das Netzwerk besteht aus den Menschen.

Self-Healing

Wenn Nodes weggehen, reroutet das Netzwerk automatisch. Das Netzwerk bleibt.

Sichtbar

Canvas-Visualisierung macht das unsichtbare Bluetooth-Netzwerk erlebbar.

Offen

Keine Vorkenntnisse nötig. Jeder kann mitmachen, lernen, bauen.

Technologie

Web Bluetooth API als Grundlage — läuft in Chrome/Edge auf Android und Desktop. Auf iOS/Safari als Simulation verfügbar.

  • Managed Flooding als Routing-Algorithmus
  • BFS-basierte Hop-Komputation
  • Relay vs Leaf Node Topologie
  • Canvas-basierte Echtzeit-Visualisierung

Kontext

Entwickelt als Workshop-Konzept für Festival Camp Weeks. Menschen erleben dezentrale Kommunikation live, verstehen die Technik, und tragen die Idee weiter.
