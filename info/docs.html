Code Guide

  • meshnet.js — Node class, state management, initNetwork, addRandomNode, BFS hops
  • canvas.js — Canvas resize, draw(), loop(), packet animation, node rendering
  • bluetooth.js — scanBluetooth(), checkBluetooth(), Web Bluetooth API
  • routing.js — sendMessage(), floodMessage(), continueFloodFrom(), receiveMessage()
  • ui.js — addMessageToUI(), updateUI(), toast(), escapeHtml()

Mesh Protocol

  • Node ID: 4-byte hex (0xXXXX)
  • Message TTL: 7 hops
  • Range: 180px canvas (~10m real)
  • Routing: Managed Flooding
  • Relay: only relay nodes forward
  • Loop prevention: route tracking
