# READ-AGENT.md — web-bluetooth.mesh

## Project Overview
Web Bluetooth Mesh Network — a decentralized peer-to-peer communication system for web browsers using the Web Bluetooth API.

## Core Artifact
`meshnet_client.html` — Single-file mesh network client with:
- Canvas-based topology visualization
- Managed flooding message routing
- Web Bluetooth device scanning
- Simulation mode
- Self-healing network demo
